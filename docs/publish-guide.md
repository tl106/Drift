## 第一阶段：酷安上架（最快1天）

### Step 1：注册酷安开发者
1. 打开 https://developer.coolapk.com
2. 用酷安账号登录（没有就先注册一个酷安账号）
3. 完成开发者认证（个人：身份证正反面照片）
4. 等待审核（通常几小时到1天）

### Step 2：准备应用截图（你做）
用手机截5张图：
1. **首页未运行** — 显示"drift"标题+琥珀色"开始"按钮
2. **首页运行中** — 红色"停止"按钮+呼吸光晕动画
3. **设置页** — 显示等待时间/渐弱时间/自动启动/后台运行
4. **记录页** — 显示至少1条检测记录（先运行一次测试生成记录）
5. **桌面Widget** — 桌面上显示Drift小组件

截图要求：
- 不要包含状态栏的个人信息（时间没关系）
- 竖屏，原始分辨率即可
- 如果手机有"截屏后编辑"功能，可以裁掉顶部状态栏

### Step 3：上传到酷安
1. 登录开发者后台
2. 点击"发布应用"
3. 上传 `app-release.apk`
4. 填写信息：

| 字段 | 填写内容 |
|------|---------|
| 应用名称 | Drift |
| 一句话简介 | 睡着自动暂停视频音乐 · 智能媒体定时器 |
| 详细介绍 | 复制 `docs/store-listing.md` 中的"详细描述"部分 |
| 应用分类 | 实用工具 |
| 应用截图 | 上传Step 2的5张图 |
| 隐私政策 | https://tl106.github.io/Drift/ |
| 开源地址 | https://github.com/tl106/Drift |

5. 提交审核
6. 等待（通常1天内）

### Step 4：发酷安首发帖
审核通过后，在酷安社区发帖：
- 复制 `docs/store-listing.md` 底部的"酷安首发帖"内容
- 发到"应用推荐"或"开发者交流"板块
- 积极回复每一条评论

## 第三阶段：主流应用商店上架（需要软著）

**前置条件：** 软著证书已到手 + APP备案已完成

### Step 0：APP备案（2024年起强制）
1. 打开 https://app.caict.ac.cn
2. 注册并登录
3. 选择"APP备案"
4. 填写信息：

| 字段 | 填写内容 |
|------|---------|
| APP名称 | Drift |
| 运营主体 | 个人 |
| 主体负责人 | 你的姓名 |
| APP分类 | 实用工具 |
| 服务类型 | 不涉及用户注册 |
| 域名 | tl106.github.io（用于隐私政策页面） |

5. 等待审批（1-2周）
6. 获得备案号

### Step 1：注册各商店开发者账号

| 商店 | 注册地址 | 需要材料 | 费用 |
|------|---------|---------|------|
| 华为 | https://developer.huawei.com | 身份证+银行卡 | 免费 |
| 小米 | https://dev.mi.com | 身份证+银行卡 | 免费 |
| OPPO | https://open.oppomobile.com | 身份证+银行卡 | 免费 |
| vivo | https://dev.vivo.com.cn | 身份证+银行卡 | 免费 |
| 应用宝 | https://open.tencent.com | 身份证+银行卡 | 免费 |

每个商店注册流程类似：注册→实名认证→等待审核（1-3天）

### Step 2：准备上架材料

每个商店都需要：

| 材料 | 说明 |
|------|------|
| APK文件 | `app-release.apk` |
| 应用图标 | 512×512 PNG (`icon_cropped.png`) |
| 应用截图 | 3-5张手机截图 |
| 应用名称 | Drift |
| 一句话简介 | 睡着自动暂停视频音乐 · 智能媒体定时器 |
| 详细描述 | 见 `docs/store-listing.md` |
| 隐私政策URL | https://tl106.github.io/Drift/ |
| 软件著作权 | 软著证书扫描件/截图 |
| APP备案号 | 备案通过后获得 |
| 版本更新说明 | 见 `docs/store-listing.md` 底部的更新日志 |

### Step 3：各商店特殊要求

**华为应用市场：**
- 应用名称必须与软著名称一致（"Drift睡眠守护软件"或简称"Drift"）
- 健康类应用建议标注"本应用非医疗器械，不具有诊断功能"
- 建议申请"绿色应用"认证（优先推荐）

**小米应用商店：**
- 可填写100字符关键词（参考store-listing.md中的关键词列表）
- 支持设置多语言元数据

**OPPO软件商店：**
- 不支持自定义关键词，优化标题和描述中的关键词
- 一句话简介尽量包含核心关键词

**vivo应用商店：**
- 流程与OPPO类似

**应用宝：**
- 腾讯系，审核相对严格
- 确保隐私政策完整且可访问

### Step 4：提交审核
- 每个商店分别提交
- 审核时间通常1-3天
- 被拒后根据拒绝原因修改后重新提交

## 版本更新流程（以后每次发新版本）

```
1. 修改 app/build.gradle.kts 中的 versionCode 和 versionName
   versionCode = 2 (递增)
   versionName = "1.1.0"

2. 编译release APK：
   ./gradlew assembleRelease

3. 上传到各应用商店，填写更新说明

4. 推送到GitHub：
   git add -A && git commit -m "v1.1.0 - xxx" && git push
```

## 重要提醒

1. **drift-release.jks 签名文件务必备份到安全的地方**（U盘/云盘）。丢失后无法更新App，只能用新签名重新发布（=新应用，丢失所有现有用户）
2. 签名密码：`drift2026`，alias：`drift`
3. 隐私政策如有修改，同步更新 `docs/privacy-policy.md`、`docs/index.html` 和GitHub
4. 每次发新版本前先在自己手机上测试
